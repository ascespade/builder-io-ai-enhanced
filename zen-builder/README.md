Zen Builder (Next.js 14 + Builder.io)

Quick start

1) Install deps
```bash
pnpm install
```

2) Copy env and set your Builder.io Public API Key
```bash
cp .env.example .env.local
# set NEXT_PUBLIC_BUILDER_API_KEY
```

3) Dev
```bash
pnpm dev
```

Open http://localhost:3010

- Dashboard: /dashboard
- Builder page: /builder/page/home
